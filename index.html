.htaccess Boilerplate - Make your website faster and more secure.

View on GitHub Download .zip Download .tar.gz

Take control of your web server! Use this boilerplate for your server's .htaccess file to make your website faster and more secure. Need to enable caching or rediect a subdirectory? Learn how with .htaccess Boilerplate. This file makes it easy to customize the behavior of your website, add password protection to folders, block IPs and bots, and so much more.

Contributors Welcome! Anyone is welcome to Help Edit and/or Add To .htaccess Boilerplate to help new webmasters provide a better web surfing experience for internet visitors.
